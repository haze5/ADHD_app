FocusGuard - ADHD专注守护 需求文档 v2.0

📋 文档信息

项目名称: FocusGuard - ADHD专注守护  

版本: v2.0

更新日期: 2024年12月

目标用户: 全谱系ADHD人群，特别关注成年确诊者、学生、职场人士的核心痛点

🎯 产品愿景升级

新的产品定位

从"专注力训练工具"升级为"**ADHD神经特质适配的操作系统**"，提供从任务管理到情绪支持的全方位解决方案。

扩展价值主张

神经多样性友好: 基于ADHD大脑运作机制设计，而非强迫适应神经典型标准

成长型陪伴: 动漫伙伴随用户一起成长，提供个性化适应

社区赋能: 连接有相似经历的ADHD用户，减少孤独感

🎪 扩展用户画像与对应功能

用户群体细分与功能映射

1. 成年后确诊者 → 认知重建模块

Plain Text
核心需求: 理解ADHD特质，重建自我认知
对应功能:
- 【认知图书馆】ADHD知识科普、成功案例
- 【特质评估】定期ADHD特质自评，可视化进步
- 【情绪日记】记录确诊后的情绪波动，提供引导

2. 学生群体 → 学术支持系统

Plain Text
核心需求: 适应自主学习，完成学术任务
对应功能:
- 【论文拆解器】专用学术任务分解模板
- 【学习计时】基于番茄工作法但更灵活
- 【考试倒计时】可视化压力转化为动力

3. 职场人士 → 职业发展套件

Plain Text
核心需求: 提升工作效率，管理职场关系
对应功能:
- 【会议准备】结构化会议前中后检查清单
- 【邮件助手】邮件模板和发送前审核
- 【职业目标】将大目标拆解为季度/月度计划

🏗️ 功能模块增强

模块一：智能任务管理系统（增强版）

1.1 多维度任务属性

Plain Text
新增任务属性:
- 价值认同度: ★★★☆☆ (用户自评该任务对长期目标的价值)
- 能量消耗: 高/中/低 (预估完成任务所需心理能量)
- 最佳执行时段: 早晨/上午/下午/晚上

ADHD科学适配:
- 根据用户能量水平和时间段推荐合适任务
- 价值度低但必要的任务提供额外激励

1.2 柔性锁机机制

Plain Text
升级为智能锁机:
- 基于RSD保护: 在用户情绪低落时适当减少锁机强度
- 学习型调整: 根据用户历史表现动态调整锁机时长
- 紧急出口: 提供有限次数的"今日豁免权"，培养责任意识

锁机期间增强功能:
- 提供正念呼吸引导
- 播放专注白噪音
- 显示用户已完成的成就

1.3 个性化难度识别

Plain Text
增强识别算法:
用户画像校准 + 文本分析 + 历史数据

难度动态调整:
- 新手期: 降低难度基准
- 成长期: 逐步增加挑战
- 波动期: 根据近期表现智能调整

模块二：灵感捕捉系统（增强版）

2.1 多模态记录

Plain Text
新增输入方式:
- 画板涂鸦: 视觉思维表达
- 语音备忘录: 快速捕捉流动想法
- 网页剪辑: 保存启发性的文章片段

智能标签:
自动基于内容打标，如"创意灵感"、"问题解决"、"自我反思"

2.2 灵感连接网络

Plain Text
灵感关联可视化:
- 显示灵感与任务间的关联网络
- 发现不同项目间的潜在联系
- 将碎片灵感整合为完整方案

模块三：动漫监督伙伴系统（情感智能版）

3.1 情感响应机制

Plain Text
情绪检测与响应:
- 通过任务完成模式和日记内容检测用户情绪状态
- RSD保护模式: 检测到拒绝敏感时提供额外支持
- 成功庆祝: 对微小成就也给予充分认可

伙伴人格成长:
- 随用户使用时间展现不同性格侧面
- 学习用户的偏好和沟通风格

3.2 增强反思引导系统

Plain Text
分层引导策略:
【Level 1: 情绪安抚】
- "看起来今天有点艰难，想聊聊发生了什么吗？"

【Level 2: 认知重构】  
- "从'我失败了'到'这个策略不太有效'，我们可以怎么调整？"

【Level 3: 解决方案】
- "基于你之前成功的经验，这次我们可以尝试..."

模块四：智能任务推荐（情境感知版）

4.1 多维度推荐引擎

Plain Text
推荐因素:
- 时间情境: 当前时段、可用时间
- 能量情境: 用户当前心理能量水平
- 优先级情境: 截止日期、重要性
- 兴趣情境: 用户最近表现出的兴趣领域

智能交替策略:
不只是难度交替，还包括:
- 认知类型交替 (分析型↔创意型)
- 社交程度交替 (独立完成↔需要协作)
- 环境要求交替 (室内固定↔户外移动)

4.2 增强任务拆解系统

Plain Text
Goblin Tools ++:
在基础拆解上增加ADHD特化功能:

- 【启动门槛评估】标识每个步骤的启动难度
- 【心流入口设计】找到任务中最可能引发兴趣的切入点
- 【阻力识别】帮助识别具体卡点并提供解决方案

⚖️ 伦理与隐私考虑

特殊保护措施

情绪数据保护: RSD相关数据本地加密存储

社区安全: 严格的欺凌和歧视内容过滤

压力调节: 防止系统本身成为压力源的设计检查

退出权利: 用户可以随时导出数据并删除账户

包容性设计

支持多种学习风格（视觉/听觉/动觉）

考虑共病情况（焦虑、抑郁等）

文化适应性设计
